# EasyMarkdown

## 简介

这是一个轻量的支持markdown文件预览和编辑的微信小程序。

## 开发相关

- 使用[uni-app](https://uniapp.dcloud.io/)框架开发；
- 使用[wemark](https://github.com/TooBug/wemark)进行markdown文本渲染。

### 支持的功能

- 预览微信聊天记录中的md文件。

- 对md文件进行编辑，支持图片上传（使用七牛云对象存储）与md语法快捷输入。

- md文件下载。
